PCSNS

This is the PCS Name Server daemon.  It provides name service translation-type
operations for a number of name address spaces.  Some of the address spaces
supported are:

+ real names (RN) to e-mail addresses (EMA)
+ real names (RN) to e-mail names (EMN)
+ INET host names to INET addresses
+ INET addresses to INET host names
+ e-mail addresses (EMAs) to real names (RNs)

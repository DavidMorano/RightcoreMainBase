NOTIFIER

This little program sends a terminal notice out according
to the specified arguments.

Synopsis:

$ notifier <type> <arg> [-f <from-agent>] [-V]

where:

<type>			type of notification
<arg>			some argument for notification
-f <from-agent>]	indiction where notification is from and originating
			agent (if available)
-V			print program version to standard-error and then exit
